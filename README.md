### Instructions

- Duration: 60-120 minutes
- Use Swift
- Use third party libraries when necessary
- You will be given detailed spec (end point and responses, etc) in person

  - Last name
  - Email
  - Password
  - Gender
  - Birth Year

Feel free to ask any questions.
